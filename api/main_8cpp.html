ScoreMode  mode

Perform a parallel, MPI‐distributed local (Smith–Waterman) alignment.

Splits X across ranks, does a two‐row DP on each chunk (with OpenMP + SIMD), finds the local max, gathers to pick the global max, then traceback on the winning rank.

Parameters
xFull first sequence.
yFull second sequence.
