# CLI Reference

### Command

```bash
./aligner --query <file> --target <file> --choice <1|2|3|4>
```

### Options

* `--query`: input FASTA
* `--target`: input FASTA
* `--choice`:

    * 1: global
    * 2: local
    * 3: LCS
    * 4: all
* `--mode`: dna | protein
* `--outdir`: output directory
* `--verbose`: debug output
